# SEP-NETs
This repository contains instruction, models, prototxt, logs file for SEP-Nets introduced in the paper "SEP-Nets: Small and Effective Pattern Networks" by Zhe Li, Xiaoyu Wang, Xutao Lv, Tianbao Yang.
